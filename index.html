safe boundary

          
What the server holds (and never sends)

          
Credentials stay server-side. The diagnostics endpoint strips any token-shaped value before responding, so this panel can be screenshotted or pasted into Slack without leaking keys.

          

            X read credential
            Read-only X API credential used for recent-search account inspiration. Optional: when missing, live reads are blocked, but drafts and templates still work.
            Private environment file
            Keep X and Postiz credentials outside the repository in a mode-0600 environment file loaded only by the server.
            Postiz API credential
            Required for real scheduling writes. Missing credential keeps scheduling safely blocked (safe-blocked, not silently dropped).
            Goro Hermes profile
            Hermes CLI profile used for real Goro generation when no Goro HTTP URL is configured. Defaults to goro.
            Port / tailnet host
            Local bind address and optional public/private proxy hostname used by the diagnostics summary block. Loopback is the safe default.
